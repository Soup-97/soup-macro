🖥️ 5. Datenschutz bei der App (SoupMacro.exe)

Die herunterladbare Applikation Soup Macro kommuniziert ausschließlich mit der GitHub Releases API um nach Updates zu suchen:

  • URL: https://api.github.com/repos/FunkelVult/soup-macro/releases/latest
  • Übertragen wird: eine Standard-HTTP-GET-Anfrage (IP-Adresse, User-Agent)
  • Zweck: Ermittlung der aktuell verfügbaren Version
  • Häufigkeit: einmalig beim Start der App

Die App protokolliert keine Tastatureingaben nach außen, sendet keine Makros oder Nutzungsdaten und kommuniziert mit keinem weiteren Server. Konfigurationsdaten werden ausschließlich lokal unter %APPDATA%\SoupMacro\config.json auf dem eigenen Rechner gespeichert.

Open Source: Der vollständige Quellcode ist auf GitHub einsehbar und überprüfbar.
