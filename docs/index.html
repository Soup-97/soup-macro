Features

Everything you need

From quick spamming to complex macros with mouse control — all in one tool.

⚡

Spam Mode

Multiple keys comma-separated, fixed or random interval (1ms–5000ms), hold mode and start delay.

F1
🎯

Tame Mode

Automatic tame sequence: Key 2 → wait → Key 1 → wait → repeat. Wait time freely configurable.

F2
🛠️

Custom Macros

Key press, wait, mouse click at X/Y and label comments. Save & load as .json.

F3
⏺️

Record & Playback

Record keyboard and mouse, play back. Speed 0.5× to 4×, mouse movements optional.

F4 / F5
🔄

Auto-Update

On startup, automatically checks for a new version — one click installs it.

Automatic
⚙️

Settings

Sound feedback, configurable panic key, auto-pause by window name, overlay, save profiles.

DE / EN
🖥️

System Tray & Overlay

Closing the window minimizes to tray. The overlay shows all mode statuses — floating & draggable.

Always active

Proper Installer

Installation

Ready in 3 Steps

No Python, no dependencies — just download and start.

1

Download the Installer

Click the download button and download SoupMacro_Setup.exe.

2

Install & Confirm SmartScreen

Windows may show a SmartScreen warning — click "More info" → "Run anyway". Normal for open-source software without expensive certificates.

3

Get Started

Soup Macro starts automatically after installation — choose a mode, adjust values, press a hotkey.
