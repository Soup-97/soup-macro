Features

Alles was du brauchst

Von schnellem Spammen bis zu komplexen Makros mit Maussteuerung — alles in einem Tool.

⚡

Spam Modus

Mehrere Tasten kommagetrennt, festes oder zufälliges Intervall (1ms–5000ms), Hold-Mode und Startverzögerung.

F1
🎯

Tame Modus

Automatische Tame-Sequenz: Taste 2 → warten → Taste 1 → warten → wiederholen. Wartezeit frei einstellbar.

F2
🛠️

Eigene Makros

Tastendruck, Warten, Mausklick an X/Y-Position und Label-Kommentare. Speichern & laden als .json.

F3
⏺️

Aufnahme & Wiedergabe

Tastatur und Maus aufzeichnen und wiedergeben. Geschwindigkeit 0.5× bis 4×, Mausbewegungen optional.

F4 / F5
🔄

Auto-Update

Beim Start wird automatisch geprüft ob eine neue Version verfügbar ist — ein Klick installiert sie.

Automatisch
⚙️

Einstellungen

Sound-Feedback, konfigurierbarer Panic-Key, Auto-Pause per Fenstername, Overlay, Profile speichern.

DE / EN
🖥️

System-Tray & Overlay

Fenster schließen minimiert ins Tray. Das Overlay zeigt den Status aller Modi — schwebend & draggable.

Immer aktiv
📦

Richtiger Installer

Installiert nach Programme, erstellt Shortcuts, Deinstallationseintrag und optional Autostart.

Installation

In 3 Schritten startklar

Kein Python, keine Abhängigkeiten — einfach herunterladen und starten.

1

Installer herunterladen

Klicke auf den Download-Button und lade SoupMacro_Setup.exe herunter.

2

Installieren & SmartScreen bestätigen

Windows zeigt ggf. eine SmartScreen-Warnung — auf „Weitere Informationen" → „Trotzdem ausführen" klicken. Das ist normal bei Open-Source-Software ohne teure Zertifikate.

3

Loslegen

Soup Macro startet automatisch nach der Installation — Modus wählen, Werte anpassen, Hotkey drücken.

⚠️ SmartScreen-Warnung: Das Programm ist Open Source und kostenlos. Da es kein teures Code-Signing-Zertifikat hat, zeigt Windows eine Warnung. Klicke auf „Weitere Informationen" und dann „Trotzdem ausführen". Die Datei wurde von VirusTotal geprüft und ist sauber — Ergebnis ansehen ↗
